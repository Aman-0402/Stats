What are Measures of Central Tendency?

Measures of Central Tendency are statistical values that represent the central or typical value of a dataset.

They answer a simple but powerful question:

"What is the most representative value of the data?"

The three most important measures are:

  • Mean
  • Median
  • Mode

Why are They Important?

üìò In Statistics:

  • Summarize large datasets using a single value
  • Foundation of descriptive statistics
  • Used in almost every statistical analysis

üìä In Data Science:

  • First step of Exploratory Data Analysis (EDA)
  • Helps understand data distribution
  • Assists in feature understanding

ü§ñ In AI / Machine Learning:

  • Used in data preprocessing
  • Handling missing values
  • Understanding data imbalance and skewness

Core Concepts & Definitions

üìä

Mean (Arithmetic Mean)

Definition: Average of all values

Mean = Œ£x / n
  • Uses all observations
  • Sensitive to outliers
üìà

Median

Definition: Middle value when data is ordered

  • If odd number of values ‚Üí middle value
  • If even number ‚Üí average of two middle values
  • Not affected by extreme values
üéØ

Mode

Definition: Most frequently occurring value

Can be:

  • Unimodal (one mode)
  • Bimodal (two modes)
  • Multimodal

‚úÖ Useful for categorical data

Real-Life / Practical Examples

üéì Example 1: Student Marks

Marks scored by 5 students: 40, 50, 55, 60, 90

  • Mean: Overall performance
  • Median: Typical student performance
  • Mode: Most common score
üè™ Example 2: Salaries in a Company

Salaries: 25k, 30k, 32k, 35k, 40k, 5 lakh

  • Mean is misleading due to very high salary
  • Median shows true central tendency

üìå This is why median is preferred for income data

Properties of Mean, Median, and Mode

Properties of Arithmetic Mean

Mathematical Properties
  • Sum of deviations from mean = 0
    Œ£(x ‚àí xÃÑ) = 0
  • Mean is rigidly defined
  • Uses all observations
  • Least affected by sampling fluctuations
  • Suitable for algebraic manipulation
Limitations
  • Highly affected by outliers
  • Not suitable for open-ended classes
  • Cannot be calculated for qualitative data

Properties of Median

Statistical Properties
  • Divides data into two equal halves
  • Positional measure
  • Not affected by extreme values
  • Can be used for open-ended distributions
  • Suitable for skewed data
Limitations
  • Not based on all values
  • Cannot be used for further algebraic analysis

Properties of Mode

Behavioral Properties
  • Value with maximum frequency
  • Can be unimodal, bimodal, or multimodal
  • Only measure applicable to nominal data
Limitations
  • May not exist
  • Not stable
  • Ignores most data values

Weighted Mean and Combined Mean

What are Weighted Mean and Combined Mean?

üìä Weighted Mean

An average where each value does not contribute equally. Some values are more important, so they are given weights.

üîó Combined Mean

An overall mean calculated by combining two or more groups having different sizes and different means.

üìå Both are extensions of the arithmetic mean and are extremely important in real-life data analysis

Why are They Important?

üìò In Statistics:

  • Used when observations have different importance
  • Frequently asked in competitive exams
  • Core topic in averages

üìä In Data Science:

  • Used in performance metrics
  • Handling group-wise averages
  • Feature engineering

ü§ñ In AI / Machine Learning:

  • Computing weighted losses
  • Dataset-level statistics
  • Evaluation metrics
