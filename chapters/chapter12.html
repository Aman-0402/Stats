What is Hypothesis Testing?

Hypothesis testing is a statistical method used to make a decision about a population based on sample data.

üëâ In simple words:

We test whether a claim is likely true or likely false using data

üß† Real-Life Intuition (Very Important)

Think like a court case ‚öñÔ∏è
  • Accused is innocent by default
  • Evidence must be strong to prove guilt

Similarly in statistics:

  • We assume something is true
  • We reject it only if evidence is strong

That "assumed true statement" is the Null Hypothesis.

Part 2: Type I & Type II Errors

üîπ Why Do Errors Exist in Hypothesis Testing?

Because we make decisions based on sample data, not the full population.

So there's always a chance of making a wrong decision.

üëâ These wrong decisions are called Type I and Type II errors

üß† Core Idea (Very Important)

In hypothesis testing, we either:

  • Reject H‚ÇÄ
