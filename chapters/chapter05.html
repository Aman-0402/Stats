Chapter 5: Measures of Dispersion

What are Measures of Dispersion?

Measures of Dispersion describe how spread out or scattered the data values are around a central value (mean/median).

While measures of central tendency tell us where the center is,

dispersion tells us how far data points move away from that center.

üìå Two datasets can have the same mean but very different variability.

Why are Measures of Dispersion Important?

üìò In Statistics:

  • Measure consistency, stability, and reliability of data
  • Essential for comparing datasets with same averages
  • Core part of descriptive statistics

üìä In Data Science:

  • Helps understand data variability
  • Required for feature scaling
  • Identifies noisy vs stable features

ü§ñ In AI / Machine Learning:

  • Used in standardization (Z-score)
  • Impacts model convergence
  • Detects outliers and variance inflation

Core Concepts & Definitions

üîπ Dispersion

  • Degree to which data values spread around the center
  • Low dispersion ‚Üí data clustered tightly
  • High dispersion ‚Üí data widely scattered

üîπ Range

  • Simplest measure of dispersion
  • Difference between maximum and minimum values
Range = Maximum ‚àí Minimum
üìå Uses only two values

üîπ Variance

  • Average of squared deviations from the mean
  • Measures overall variability
Population Variance
Sample Variance
üìå Squaring avoids negative deviations

üîπ Standard Deviation

  • Square root of variance
  • Expressed in same units as data
Standard Deviation Formula
üìå Most widely used dispersion measure

Real-Life / Practical Examples

üéì Example 1: Student Marks

Class A: 50, 52, 53, 54, 55

Class B: 30, 45, 55, 70, 90

  • Both may have similar mean
  • Class B has higher dispersion
üè™ Example 2: Product Quality Control
  • Low SD ‚Üí consistent quality
  • High SD ‚Üí defective products likely

Interquartile Range (IQR)

What is Interquartile Range (IQR)?

The Interquartile Range (IQR) is a measure of dispersion that shows how spread out the middle 50% of the data is.

It focuses only on the central portion of the dataset, ignoring extreme values (outliers).

IQR = Q3 ‚àí Q1
Interquartile Range Visualization
Where:
‚Ä¢ Q1 = First quartile (25th percentile)
‚Ä¢ Q3 = Third quartile (75th percentile)

Why is IQR Important?

üìò In Statistics:

  • Robust measure of spread
  • Less affected by outliers than range or variance
  • Commonly used with boxplots

üìä In Data Science:

  • Outlier detection
  • Feature spread analysis
  • Robust EDA for skewed data

ü§ñ In AI / Machine Learning:

  • Removing extreme values before training
  • Improving model stability
  • Used in robust preprocessing pipelines

Core Concepts & Definitions

üîπ Quartiles

  • Q1 (First Quartile) ‚Üí 25% data below this value
  • Q2 (Second Quartile / Median) ‚Üí 50% data below
  • Q3 (Third Quartile) ‚Üí 75% data below

üîπ Interquartile Range (IQR)

  • Measures spread of middle 50%
  • Formula: IQR = Q3 ‚àí Q1

üîπ Outlier Detection (IQR Method)

Lower Bound
Q1 ‚àí 1.5√óIQR
Upper Bound
Q3 + 1.5√óIQR
Values outside these bounds are considered outliers.

Real-Life / Practical Examples

üéì Example 1: Student Marks

Marks: 35, 40, 45, 50, 55, 60, 90

  • Extreme high value (90) exists
  • Mean & SD are affected
  • IQR focuses on typical students
üè™ Example 2: House Prices
  • A few luxury houses skew the data
  • IQR reflects normal market range

Mathematical / Numerical Example (Step-by-Step)

üî¢ Given Data (Sorted): 10, 12, 14, 16, 18, 20, 22, 24

üîπ Step 1: Find Median (Q2)

Middle values = 16 and 18

Q2 = (16 + 18) / 2 = 17

üîπ Step 2: Find Q1

Lower half: 10, 12, 14, 16

Q1 = (12 + 14) / 2 = 13

üîπ Step 3: Find Q3

Upper half: 18, 20, 22, 24

Q3 = (20 + 22) / 2 = 21

üîπ Step 4: Compute IQR

IQR = 21 ‚àí 13 = 8

üîπ Step 5: Outlier Limits

  • Lower = 13 ‚àí 1.5(8) = 1
  • Upper = 21 + 1.5(8) = 33

üìå Any value < 1 or > 33 is an outlier

Coefficient of Variation (CV)

What is the Coefficient of Variation?

The Coefficient of Variation (CV) is a relative measure of dispersion that shows how much variability exists relative to the mean.

Instead of telling spread in absolute units (like SD), CV tells spread in percentage terms.

CV = (Standard Deviation / Mean) √ó 100

Why is CV Important?

üìò In Statistics:

  • Compares consistency between datasets
  • Useful when means are different
  • Widely used in exams and theory questions

üìä In Data Science:

  • Comparing variability of features with different scales
  • Evaluating data stability
  • Feature reliability analysis

ü§ñ In AI / Machine Learning:

  • Selecting stable features
  • Comparing model metrics across datasets
  • Performance consistency checks

Core Concepts & Definitions

üîπ Absolute vs Relative Dispersion

Standard Deviation
Absolute variability (units matter)
Coefficient of Variation
Relative variability (unit-free)

üîπ Key Properties of CV

  • Expressed in percentage
  • Lower CV ‚Üí More consistent data
  • Higher CV ‚Üí More variability / risk
  • Meaningful only when mean > 0

Real-Life / Practical Examples

üéì Example 1: Two Classes' Marks
Class Mean Standard Deviation
A 60 6
B 80 10
  • SD alone is misleading
  • CV compares relative spread
üè™ Example 2: Investment Risk
  • Lower CV ‚Üí Safer investment
  • Higher CV ‚Üí Riskier returns

Mathematical / Numerical Example

üî¢ Step-by-Step Calculation

For Class A:

CVA = (6 / 60) √ó 100 = 10%

For Class B:

CVB = (10 / 80) √ó 100 = 12.5%

üìå Interpretation:

  • Class A is more consistent
  • Class B has greater relative variation

Interpretation Guide

CV Value Interpretation
< 10% Very consistent
10% ‚Äì 20% Moderate variation
> 20% High variation
(Rule of thumb ‚Äì may vary by domain)

Comparison with Other Measures
