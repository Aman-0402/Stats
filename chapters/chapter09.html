What is Probability?

Probability measures how likely an event is to happen.

It always lies between:

0 ‚â§ P(event) ‚â§ 1
  • 0 ‚Üí impossible
  • 1 ‚Üí certain

Part 1: Classical Probability

Classical Probability (Theoretical Probability)

‚úÖ Definition

Classical probability is used when all outcomes are equally likely

üìê Formula

P(E) = Number of favorable outcomes / Total number of possible outcomes

Example 1: Tossing a Fair Coin

ü™ô Coin Toss

Sample Space (S):

S = {H, T}

  • Total outcomes = 2
  • Favorable outcomes for Head = 1
P(H) = 1/2
üí° Since the coin is fair, both outcomes are equally likely ‚Üí classical probability applies

üé≤ Example 2: Rolling a Fair Die

Sample Space

S = {1, 2, 3, 4, 5, 6}

Q1: Probability of getting an even number?

  • Favorable outcomes = {2, 4, 6} ‚Üí 3
  • Total outcomes = 6
P(Even) = 3/6 = 1/2

Q2: Probability of getting a number greater than 4?

  • Favorable outcomes = {5, 6} ‚Üí 2
P(Number > 4) = 2/6 = 1/3

üß† Key Assumptions

Classical probability works ONLY IF:

  1. All outcomes are known
  2. All outcomes are equally likely
  3. Situation is theoretical, not based on experiments

‚ùå Not suitable when:

  • Data comes from real-life observations
  • Outcomes are not equally likely

Part 2: Empirical Probability (Experimental Probability)

üîπ What is Empirical Probability?

Empirical probability is based on actual experiments, observations, or real data, not theory.

üëâ It answers:

"What happened in real life?"

üìê Formula

P(E) = Number of times event occurred / Total number of trials
This is also called relative frequency

Example 1: Coin Toss Experiment

ü™ô Real Coin Toss

Suppose you toss a coin 100 times and observe:

  • Heads = 56 times
  • Tails = 44 times

Probability of getting Head:

P(H) = 56/100 = 0.56

Notice:

  • Theoretical (classical) value = 0.5
  • Empirical value = 0.56

‚û°Ô∏è Because this is based on real outcomes, not assumptions

Example 2: Real-Life Student Example

üéì Class Performance

In a class:

  • Total students = 50
  • Students who passed = 42
P(Pass) = 42/50 = 0.84

‚úîÔ∏è This is empirical probability, because it comes from observed data

‚ö° Important!

As the number of trials increases,

üëâ Empirical probability approaches classical probability

This idea is called the Law of Large Numbers

Python Implementation

Let's simulate a coin toss üëá

Python 
import numpy as np

# Simulate 1000 coin tosses (0 = Tail, 1 = Head)
tosses = np.random.choice([0, 1], size=1000)

# Empirical probability of head
prob_head = np.sum(tosses == 1) / len(tosses)
prob_head
0.497
‚û°Ô∏è Close to 0.5, but not exactly ‚Äî because it's empirical

Part 3: Subjective Probability

What is Subjective Probability?

Subjective probability is based on personal judgment, experience, belief, or opinion, not on formulas or experiments.

üëâ It answers:

"What do I think will happen?"

üìê Is there a formula?

‚ùå No fixed formula like classical or empirical probability.

It depends on:

  • Past experience
  • Expertise
  • Available information
  • Intuition

üåßÔ∏è Example 1: Weather Forecast

Weather Prediction

A meteorologist says:

"There is a 70% chance of rain tomorrow."

‚úîÔ∏è This is subjective probability because:

  • It's based on experience + models + judgment
  • Not a simple equally-likely outcome
  • Not a repeated experiment by you

‚öΩ Example 2: Sports Prediction

Sports Match

You say:

"India has a 60% chance of winning this match."

Why subjective?

  • Based on team form, pitch, injuries
  • Different experts may give different probabilities

üíº Example 3: Business / ML Thinking

Startup Success

A startup founder says:

"There's an 80% chance our app will succeed."

This is:

  • Not theoretical
  • Not experimental
  • Based on belief + market knowledge

üß† Why Subjective Probability Matters

In Data Science & AI:

  • Prior beliefs are used in Bayesian Statistics
  • Human judgment influences:
    • Risk analysis
    • Decision-making systems
    • Expert systems

üëâ Later, you'll see how subjective beliefs + data = Bayesian probability

Part 4: Rules of Probability

üîπ Why Rules of Probability Matter?

Rules help us:

  • Combine events
  • Avoid double counting
  • Solve real exam & ML problems

Think of them as grammar rules of probability.

üß© Rule 1: Probability of an Event Lies Between 0 and 1

0 ‚â§ P(A) ‚â§ 1
  • 0 ‚Üí Impossible
  • 1 ‚Üí Certain

Example:

  • P(getting 7 on a die) = 0
  • P(sun rises tomorrow) ‚âà 1

üß© Rule 2: Sum of Probabilities of All Outcomes = 1

If S is the sample space:

P(S) = 1
Example: Fair Die

P(1) + P(2) + P(3) + P(4) + P(5) + P(6) = 1

üß© Rule 3: Complement Rule

üìê Formula

P(Ac) = 1 ‚àí P(A)
Where:
‚Ä¢ Ac = "NOT A"
Example

Probability of rain today = 0.3

P(No rain) = 1 ‚àí 0.3 = 0.7

‚úîÔ∏è Very useful when direct calculation is hard

üß© Rule 4: Addition Rule

Case 1: Mutually Exclusive Events

(Events that cannot happen together)

P(A ‚à™ B) = P(A) + P(B)
Example

On a die:

  • A = getting 1 ‚Üí 1/6
  • B = getting 6 ‚Üí 1/6

P(1 or 6) = 1/6 + 1/6 = 1/3

Case 2: Not Mutually Exclusive

(Events can happen together)

P(A ‚à™ B) = P(A) + P(B) ‚àí P(A ‚à© B)
Example

From a class:

  • P(Student plays Cricket) = 0.6
  • P(Student plays Football) = 0.5
  • P(Plays both) = 0.2

P(C or F) = 0.6 + 0.5 ‚àí 0.2 = 0.9

üëâ Subtraction avoids double counting

üß† Visual Intuition (Venn Diagram Logic)

  • Overlapping area = P(A ‚à© B)
  • Union = total shaded area

üß™ Python Example (Addition Rule)

Python 
P_A = 0.6
P_B = 0.5
P_A_and_B = 0.2

P_A_or_B = P_A + P_B - P_A_and_B
P_A_or_B
0.9

Part 5: Independent & Mutually Exclusive Events

First, Understand "Events"

An event is any outcome or group of outcomes.

Examples:

  • Getting Head
  • Rolling a 6
  • Student passing exam

üîµ A. Mutually Exclusive Events

‚úÖ Definition

Two events are mutually exclusive if they cannot occur at the same time

P(A ‚à© B) = 0
üé≤ Example 1: Die Roll
  • A = getting 2
  • B = getting 5

You cannot get 2 and 5 together in one roll.

‚úîÔ∏è Mutually exclusive

P(A ‚à© B) = 0

üß† Key Property

If events are mutually exclusive:

P(A ‚à™ B) = P(A) + P(B)
‚ùå Not Mutually Exclusive Example
  • A = getting even number
  • B = getting number greater than 3

Overlap = {4, 6} ‚Üí not mutually exclusive

üü¢ B. Independent Events

‚úÖ Definition

Two events are independent if the occurrence of one does NOT affect the other

P(A ‚à© B) = P(A) √ó P(B)
ü™ô Example 2: Tossing Two Coins
  • A = Head on first coin ‚Üí 1/2
  • B = Head on second coin ‚Üí 1/2

First coin doesn't affect second coin.

P(A ‚à© B) = 1/2 √ó 1/2 = 1/4

‚úîÔ∏è Independent

üé≤ Example 3: Coin Toss + Die Roll

Coin result has no effect on die

‚úîÔ∏è Independent events

üö® VERY IMPORTANT CONFUSION (Exam Favorite)

‚ùå Mutually Exclusive ‚â† Independent

Case Mutually Exclusive Independent
Can occur together? ‚ùå No ‚úÖ Yes
Intersection 0 P(A) √ó P(B)
Relationship Disjoint Unrelated
üëâ They can NEVER be both at the same time (except trivial zero-probability cases)

üß† Trick to Remember

  • Mutually Exclusive ‚Üí No overlap
  • Independent ‚Üí No influence

# Probability of getting Head on first coin

# Probability of getting Head on second coin
P_B = 0.5

# Independent events
P_A_and_B = P_A * P_B

0.25

Exam-Level Numerical Problems on Probability

‚úÖ Problem 1: Classical Probability

Q1. A fair die is thrown once. Find the probability of getting

(a) a prime number

(b) a number divisible by 3

‚úçÔ∏è Solution:

Sample space: S = {1, 2, 3, 4, 5, 6}, n(S) = 6

(a) Prime numbers = {2, 3, 5}

Favorable outcomes = 3

P(Prime) = 3/6 = 1/2

(b) Numbers divisible by 3 = {3, 6}

Favorable outcomes = 2

P(Divisible by 3) = 2/6 = 1/3
‚úÖ Problem 2: Complement Rule

Q2. The probability that a student passes an exam is 0.78. Find the probability that the student fails.

‚úçÔ∏è Solution:

Let: P(Pass) = 0.78

Using complement rule:

P(Fail) = 1 ‚àí P(Pass)

P(Fail) = 1 ‚àí 0.78

P(Fail) = 0.22

Answer: 0.22

‚úÖ Problem 3: Empirical Probability

Q3. In 200 tosses of a coin, heads appeared 110 times. Find the probability of getting a head.

‚úçÔ∏è Solution:

P(H) = Number of heads / Total tosses

P(H) = 110 / 200

P(H) = 0.55

‚úîÔ∏è Answer: 0.55

‚úÖ Problem 4: Mutually Exclusive Events

Q4. A card is drawn from a deck of 52 cards. Find the probability that the card is a king or a queen.

‚úçÔ∏è Solution:

  • Number of Kings = 4
  • Number of Queens = 4

These events are mutually exclusive (a card can't be both).

P(K ‚à™ Q) = P(K) + P(Q)

P = 4/52 + 4/52 = 8/52

P = 0.1538
‚úÖ Problem 5: Not Mutually Exclusive Events

Q5. From a class:

  • 60% students like Mathematics
  • 50% like Physics
  • 20% like both

Find the probability that a student likes Mathematics or Physics.

‚úçÔ∏è Solution:

Given:

P(M) = 0.6, P(P) = 0.5, P(M ‚à© P) = 0.2

Using addition rule:

P(M ‚à™ P) = P(M) + P(P) ‚àí P(M ‚à© P)

P = 0.6 + 0.5 ‚àí 0.2

P = 0.9

‚úîÔ∏è Answer: 0.9

‚úÖ Problem 6: Independent Events

Q6. Two coins are tossed simultaneously. Find the probability of getting two heads.

‚úçÔ∏è Solution:

  • P(H on first coin) = 1/2
  • P(H on second coin) = 1/2

Coins are independent.

P(HH) = 1/2 √ó 1/2

P(HH) = 1/4

‚úîÔ∏è Answer: 1/4

‚úÖ Problem 7: Mixed Concept

Q7. A number is chosen at random from {1, 2, 3, 4, 5, 6, 7, 8}. Find the probability that the number is even or prime.

‚úçÔ∏è Solution:

  • Even numbers = {2, 4, 6, 8} ‚Üí 4
  • Prime numbers = {2, 3, 5, 7} ‚Üí 4
  • Common (both even & prime) = {2} ‚Üí 1

Using addition rule:

P(E ‚à™ P) = 4/8 + 4/8 ‚àí 1/8

P(E ‚à™ P) = 7/8

‚úîÔ∏è Answer: 7/8
