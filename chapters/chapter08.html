Introduction to Regression

Regression is one of the most important techniques in Statistics, Data Science, Machine Learning, and AI. It helps us understand relationships between variables and predict future values.

What is Regression?

Regression is a statistical method used to:

  • Study the relationship between variables
  • Predict the value of one variable based on another
  • Measure how much change in one variable affects another

üìå Simple Definition

Regression tells us how a dependent variable changes when an independent variable changes

Variables in Regression

Every regression model has two main variables:

üîπ Independent Variable (X)
  • Also called predictor, input, or explanatory variable
  • It causes or influences change
  • Example: Study hours, temperature, experience
üîπ Dependent Variable (Y)
  • Also called response or output variable
  • It depends on X
  • Example: Marks, ice cream sales, salary
üìå Example
  • X = Hours studied
  • Y = Exam marks

Regression helps answer:

üëâ If study hours increase, how will marks change?

Why Do We Need Regression?

Regression is used to:

  • Make predictions
  • Identify strength and direction of relationships
  • Perform trend analysis
  • Support decision making

üí° Real-Life Examples

üè† Predicting house prices from area

üíº Estimating salary from experience

üìä Forecasting sales based on advertising cost

üå°Ô∏è Predicting temperature trends

üìå For beginners, we always start with Simple Linear Regression

Simple Linear Regression

üîπ Model Form

Y = a + bX
Where:
‚Ä¢ Y = Dependent variable
‚Ä¢ X = Independent variable
‚Ä¢ a = Intercept (value of Y when X = 0)
‚Ä¢ b = Slope (rate of change)

üîπ Interpretation

  • Slope (b) tells how much Y changes when X increases by 1 unit
  • Intercept (a) is the starting value of Y
üìå Example

Marks = 20 + 5 √ó Study Hours

  • Each extra study hour increases marks by 5
  • Without study, marks = 20

Regression Line (Line of Best Fit)

  • The regression line best represents the data
  • It minimizes the sum of squared errors (Least Squares Method)
  • Errors = Actual Y ‚àí Predicted Y (called residuals)

üìå A good regression line:

  • Has small residuals
  • Closely follows the data trend

Assumptions of Linear Regression (Intro Level)

  1. Linear relationship between X and Y
  2. Errors are independent
  3. Errors have constant variance (Homoscedasticity)
  4. Errors are normally distributed
(We'll study these in detail later)

Regression vs Correlation

Correlation Regression
Measures strength of relationship Predicts values

No dependent variable Has dependent variable
Range: ‚àí1 to +1 No fixed range

üìå Correlation ‚Üí Relationship

üìå Regression ‚Üí Prediction

Least Squares Method (Heart of Regression)

The Least Squares Method is the core idea behind regression. It explains how we get the best-fitting line for given data.

What is the Least Squares Method?

The Least Squares Method is a technique used to:

  • Find the best fitting regression line
  • Minimize the errors (residuals) between actual and predicted values

üìå Simple Definition

The best regression line is the one for which the sum of squared errors is minimum

What Are Errors (Residuals)?

For each data point:

Error (Residual) = Yactual ‚àí Ypredicted

Since errors can be positive or negative, we square them to avoid cancellation.

Squared Error = (Y ‚àí ≈∂)¬≤

Why "Squares" and Not Absolute Values?

No cancellation Positive & negative errors don't cancel
Penalizes large errors Bigger mistakes get more weight
Mathematical simplicity Easier differentiation
üìå That's why it's called Least Squares, not Least Errors

Python Implementation of Simple Linear Regression

We'll implement Simple Linear Regression in Python step by step, exactly matching the statistics formulas + data science practice.

Using NumPy (Pure Statistics Approach)

This follows the Least Squares formulas you learned.

üìå Example Data

Python 
import numpy as np

# Independent variable (X) ‚Äì Study hours
X = np.array([1, 2, 3, 4, 5])

# Dependent variable (Y) ‚Äì Marks
Y = np.array([40, 50, 60, 70, 80])

Step 1: Calculate Required Values

Python 
n = len(X)

sum_x = np.sum(X)
sum_y = np.sum(Y)
sum_xy = np.sum(X * Y)
sum_x2 = np.sum(X ** 2)

Step 2: Compute Slope (b)

Python 
b = (n * sum_xy - sum_x * sum_y) / (n * sum_x2 - sum_x ** 2)
b
10.0

Step 3: Compute Intercept (a)

Python 
x_mean = np.mean(X)
y_mean = np.mean(Y)

a = y_mean - b * x_mean
a
30.0

‚úÖ Final Regression Equation

Python 
print(f"Regression Equation: Y = {a:.2f} + {b:.2f}X")
Regression Equation: Y = 30.00 + 10.00X

Prediction Using the Model

Python 
# Predict marks for 6 study hours
x_new = 6
y_pred = a + b * x_new
y_pred
90.0

Visualization

Python 
import matplotlib.pyplot as plt

plt.scatter(X, Y, label="Actual Data")
plt.plot(X, a + b * X, linestyle='--', label="Regression Line")
plt.xlabel("Study Hours")
plt.ylabel("Marks")
plt.legend()
plt.show()

  • üìå Line ‚Üí least squares best-fit line
